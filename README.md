Study Time Analyzer & Focus Insight Tool

The Study Time Analyzer & Focus Insight Tool is a web-based application designed to help students track their study sessions, analyze study patterns, and gain insights to improve focus and consistency.
It provides real-time study tracking, daily and weekly analytics, and intelligent insights such as streaks, consistency, and most/least studied subject

Objectives
Track study time accurately with live timers

Analyze daily and weekly study performance

Identify study consistency and streaks

Highlight most and least studied subjects

Help students improve focus and time management

Technologies Used
Frontend
HTML5
CSS3
Bootstrap 5
JavaScript
Chart.js (Bar, Line, Donut charts)

Backend
PHP
MySQL 

Tools
XAMPP
VS Code

Architecture

The project follows a three-tier architecture:
Client Layer: Browser (UI + Charts)
Server Layer: PHP backend
Database Layer: MySQL


How to Run the Project

Install XAMPP
Install php
Start Apache and MySQL
Place the project folder inside:
C:\xampp\htdocs\

Create a database:
study_analyzer1

Configure database in:
db/config.php


Open browser and go to:
http://localhost/study_analyzer
